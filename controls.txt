d============= Movement =============

Move Left -> A
Move Right -> D
Move Slowly / Fall -> S
Jump -> Space

=========== Tile Editing ===========

Place Tile -> MB1 with block selected
Remove Tile -> MB1 with pickaxe selected

Place Wall Tile -> Shift + MB1 with block selected
Remove Wall Tile -> Shift + MB1 with pickaxe selected

============ Inventory =============

Toggle Inventory -> Escape
Select Hotbar Items -> 0-9 or Scroll Wheel 

============= Cheats ===============

Kill All Enemies -> Shift + X
Spawn Enemy -> Shift + F
Respawn -> R
Set Spawn at Player -> Shift + R
Reverse Gravity -> Shift + G
Randomize Selected Item Prefix -> Shift + C

============ Settings ==============

Toggle Experimental Lighting -> L
Toggle Background -> Shift + B
Toggle UI -> U
Toggle Smooth Camera -> J
Toggle Hitboxes -> H

Music Volume Up -> Shift + Up Arrow
Music Volume Down -> Shift + Down Arrow
SFX Volume Up -> Shift + Right Arrow
SFX Volume Down -> Shift + Left Arrow

========== General/Misc =============

Take World Snapshot -> Shift + T
Get Tile ID at Mouse -> P
Get Wall ID at Mouse -> Shift + P
Random Prompt Test -> Shift + V
Quit Game -> Windows Close Cross / Open Inventory + Bottom Right Quit button
